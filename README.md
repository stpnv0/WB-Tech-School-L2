# L2


## Описание проектов

### 8. NTP Time Client

Простой клиент для получения точного времени от NTP-сервера. Использует библиотеку `github.com/beevik/ntp` для подключения к `pool.ntp.org` и вывода текущего времени.

### 9. String Unpacker

Утилита для распаковки строк, сжатых с помощью простого алгоритма RLE. Поддерживает экранирование символов обратной косой чертой и повторение символов через цифры. Например, строка `\12` распакуется как `11`, а `a4` — как `aaaa`.

### 10. WB Sort

Клон утилиты `sort` для сортировки строк из файла или стандартного ввода. Поддерживает сортировку по числовым значениям с флагом `-n`, игнорирование ведущих пробелов с `-b` и проверку сортировки с `-c`. Реализует алгоритм быстрой сортировки.

### 11. Anagram Finder

Программа для поиска анаграмм в наборе слов. Группирует слова, состоящие из одних и тех же букв, и выводит только те группы, где содержится минимум два слова. Например, "пятак", "пятка" и "тяпка" будут определены как анаграммы.

### 12. WB Grep

Упрощённый аналог утилиты `grep` для поиска подстрок в тексте. Поддерживает ключи `-A` (показать N строк после совпадения), `-B` (показать N строк до совпадения), `-C` (показать N строк вокруг совпадения), `-c` (только количество совпадений) и `-i` (игнорировать регистр).

### 13. WB Cut

Аналог утилиты `cut` для вырезания колонок из строк по разделителю. Поддерживает ключи `-d` (разделитель полей), `-f` (номера полей) и `-s` (не выводить строки без разделителя). Позволяет выбрать конкретные колонки из табличных данных.

### 14. OR Channel Pattern

Реализация паттерна `or` для каналов в Go. Функция принимает несколько каналов и возвращает один канал, который закрывается при закрытии любого из входных каналов. Реализовано двумя способами: через рекурсию и через `sync.Once`.

### 15. Simple Shell

Простейшая командная оболочка на Go. Поддерживает выполнение внешних команд, встроенные команды (`cd`, `pwd`, `echo`, `kill`, `ps`, `exec`, `fork`, `exit`) и конвейеры с пайпами (`|`). Обеспечивает базовую интерактивность с пользователем.

### 16. WB Wget

Клон утилиты `wget` для загрузки файлов и веб-страниц по URL. Поддерживает рекурсивную загрузку с указанием глубины `-r`, ограничение на количество потоков `-workers`, сохранение в директорию `-o` и настройку таймаута. Включает фильтрацию файлов по расширению.

### 17. Go Telnet

Простая реализация Telnet-клиента на Go. Подключается к указанному хосту и порту по протоколу TCP и обеспечивает двунаправленную передачу данных между клиентом и сервером. Поддерживает настройку таймаута подключения и graceful shutdown.

### 18. Calendar API

RESTful API для управления календарём событий на базе фреймворка Gin. Поддерживает CRUD-операции над событиями (создание, обновление, удаление, получение), валидацию данных и логирование.
